	Conversor De Monedas

Este programa hecho con java en IntelliJ haciendo el uso de una API y Gson fue creado para hacer la conversion del tipo de un moneda en base al monto que le asigne el usuario a este
simplemente hay que corre el programa y elegir la conversion de moneda que le interese al usuario averiguar y poner el monto y ya el programa le mostrara el resultado y seguir asi hasta que este desee salir.
